# Web 管理界面功能说明

## 界面预览

Web 管理界面采用现代化设计，提供直观的监控管理体验。

## 主要功能

### 1. 仪表盘概览

页面顶部显示实时统计数据：

- **在线监控数**: 当前状态为"在线"的监控数量
- **离线监控数**: 当前状态为"离线"的监控数量
- **总监控数**: 系统中配置的监控总数
- **平均响应时间**: 所有监控的平均响应时间

### 2. 监控列表

显示所有已配置的监控项，包括：

| 列名 | 说明 |
|------|------|
| 名称 | 监控项名称 |
| 类型 | 监控类型（HTTP/HTTPS/TCP/UDP/DNS） |
| 地址 | 监控目标地址和端口 |
| 状态 | 当前状态（在线/离线/未知） |
| 响应时间 | 最近一次检查的响应时间 |
| 正常运行时间 | 30天正常运行时间百分比（带可视化进度条） |
| 操作 | 编辑和删除按钮 |

#### 状态标识

- 🟢 **在线**: 监控目标正常响应
- 🔴 **离线**: 监控目标无法访问
- 🟡 **未知**: 尚未完成首次检查

### 3. 添加/编辑监控

点击"添加监控"按钮或列表中的编辑按钮，打开监控配置表单。

#### 表单字段

- **名称** (必填): 监控项的显示名称
- **类型** (必填): 选择监控类型
  - HTTP: 监控 HTTP 网站
  - HTTPS: 监控 HTTPS 网站
  - TCP: 监控 TCP 端口
  - UDP: 监控 UDP 端口
  - DNS: 监控 DNS 解析
- **地址** (必填): 监控目标地址（域名或IP）
- **端口**: 端口号（某些类型自动填充默认端口）
- **检查间隔**: 检查频率（秒），默认 60 秒
- **启用监控**: 是否启用此监控

#### 智能默认端口

根据选择的监控类型，系统会自动填充默认端口：

- HTTP: 80
- HTTPS: 443
- TCP/UDP: 留空，需手动填写
- DNS: 无需端口

### 4. IP 地理位置查询

右侧面板提供 IP 地址归属地查询功能。

#### 使用方法

1. 在输入框中输入 IP 地址（例如：8.8.8.8）
2. 点击"查询"按钮
3. 查看查询结果

#### 查询结果

查询结果会显示以下信息：

- **IP 地址**: 查询的 IP 地址
- **国家**: IP 所属国家
- **地区**: IP 所属地区/省份
- **城市**: IP 所属城市
- **ISP**: 网络服务提供商
- **坐标**: 纬度和经度

### 5. 自动刷新

- 系统每 30 秒自动刷新监控状态
- 点击"刷新"按钮可手动立即刷新
- 自动刷新时会有 Toast 提示

### 6. 响应式设计

界面采用响应式设计，适配不同屏幕尺寸：

- **桌面端**: 双栏布局，监控列表和 IP 查询并排显示
- **平板/移动端**: 单栏布局，内容垂直排列

## 交互细节

### 模态框操作

- 点击"添加监控"打开新建表单
- 点击监控列表中的编辑图标打开编辑表单
- 点击表单外的背景区域或按 ESC 键关闭表单
- 表单验证失败会显示错误提示

### Toast 通知

操作成功或失败时，右下角会弹出 Toast 通知：

- ✅ 成功操作：绿色边框
- ❌ 失败操作：红色边框
- 3 秒后自动消失

### 确认对话框

删除监控时会弹出确认对话框，防止误操作。

## 使用建议

1. **合理设置检查间隔**: 建议不少于 30 秒，避免过于频繁的检查
2. **监控命名**: 使用清晰的名称，方便识别监控目标
3. **定期查看**: 定期查看正常运行时间百分比，了解服务稳定性
4. **IP 查询缓存**: IP 查询结果会被缓存，提高查询速度

## 技术特性

- **纯原生实现**: 使用原生 JavaScript，无需额外框架
- **现代 CSS**: 采用 CSS Grid 和 Flexbox 布局
- **图标支持**: 使用 Font Awesome 图标库
- **异步请求**: 所有 API 调用均使用异步方式
- **错误处理**: 完善的错误处理和用户提示

## 浏览器兼容性

支持所有现代浏览器：

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
